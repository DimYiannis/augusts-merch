## Project Setup

### Clone the repo
```sh
git clone https://github.com/DimYiannis/augusts-merch.git
```
### Install NPM packages
```sh
npm install
```


### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
